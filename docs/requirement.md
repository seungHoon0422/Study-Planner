# Study Planner 개요

SQLite 를 사용해서 데이터를 가볍가 저장
하나의 페이지에 달력이 보이는 구조.
전체적으로 깔끔한 UI로 구성 (핑크색 기반 다양한 파스텔 색상으로 구성)

날짜를 클릭하면 해당 날짜의 과제 관리를 할 수 있는 화면으로 구성
날짜 우측 상단에 + 버튼을 통해 과제 등록하는 모달 생성
입력 내용은 제목, description, 시간 설정 가능
시간 설정은 시작시간(default : 오후 6시), 걸리는 시간으로 구성
걸리는 시간은 30분 간격으로 4시간 까지 드롭다운으로 선택할 수 있게 구성
